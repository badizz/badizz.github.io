Graduation Project

● The project was developed for tracking our pets both inside and outside the home. It can track the activities of pets (running, walking, resting and sleeping). It keeps track of its current location and past locations.
● Within the scope of the project: Arduino, GSM-GPS shield and sensors for activities on the device side; HTML / PHP on the web application side, Ionic/Javascript on the mobile application side, MySQL was used as database.
● The project received the first prize in the department.
